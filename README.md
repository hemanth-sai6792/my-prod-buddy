# 🕒 Productivity Desk

A simple personal dashboard combining:
- 📅 Live Clock & Calendar
- ⏳ Countdown Task Tracker
- 📝 Daily Notes Section
- 📚 Knowledge Center (coming soon)
- 🎯 Motivational Time Tracker with animated progress
